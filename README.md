 E-Commerce Backend
 
This is a backend application developed using Java, Spring Boot, Spring Data JPA, and MySQL. It provides REST APIs for managing products, searching products, and uploading product images.
The project follows a layered architecture using Controller, Service, and Repository components.

## Features

- Product CRUD operations
- Product image upload
- Product search
- Spring Security configuration
- MySQL database integration
- Exception handling

## Technologies

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Git
